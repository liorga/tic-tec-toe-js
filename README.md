# tic-tec-toe-js

# steps to use this app

# 1.

clone this repo to your local system. using git clone https://github.com/liorga/tic-tec-toe-js.git
command from your terminal

# 2.

cd into tic-tac-toe-js folder.

# 3.

use npm install.

# 4.

cd into server folder.

# 5.

start the app using node server.js

# 6.

open 2 different browsers on lochalhost:8080 and enjoy the game :)
